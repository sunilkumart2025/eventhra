Description

Reverse Coding is a logic-based challenge where you're given the output — and must figure out the code behind it. Think backwards. Decode patterns. Crack the logic.

Detailed Information

Reverse Coding is a brain-teasing challenge designed to test your analytical and problem-solving skills. Unlike traditional coding problems, here you're presented with the final output, and your task is to work backward to determine the underlying logic or code that could produce it. Whether it's discovering mathematical patterns, decoding sequences, or reverse-engineering functions, Reverse Coding encourages you to think differently, analyze deeply, and solve smartly. It's not just about writing code — it's about understanding the logic behind it. Perfect for those who enjoy puzzles, logic games, and thinking outside the box!

2 Rounds Round 1: 45 mins Round 2: 45 mins

Event Rules

  • All rounds are time-bound; no extra time will be provided.
  • Output will be given - teams must reverse-engineer the logic or code behind it.
  • Round 1 is an elimination round - only top teams proceed to the next rounds.
  • No use of mobile phones, internet, or external help during the rounds.
